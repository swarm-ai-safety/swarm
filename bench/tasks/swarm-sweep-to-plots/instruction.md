# Task: Generate Plots from Sweep Data

Generate visualizations from the sweep CSV at /root/data/sweep_results.csv. Create at minimum: a grouped bar chart of welfare by tax rate, a box plot of welfare distribution, and an agent payoff comparison chart. Save all plots to /root/output/plots/.

## Requirements

- Input: `/root/data/sweep_results.csv` (pre-loaded, 12 rows)
- Output directory: `/root/output/plots/`
- Minimum 3 PNG files in the plots directory
- Each PNG must be >5KB (not empty placeholder)
- Use matplotlib for plotting

## Environment

- Python 3.12 with matplotlib, pandas, numpy pre-installed
- Sweep CSV is pre-loaded at `/root/data/sweep_results.csv`
