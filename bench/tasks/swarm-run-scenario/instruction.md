# Task: Run Baseline Scenario

Run the baseline scenario with seed 42 for 10 epochs and 10 steps. Export history.json and CSV metrics to /root/output/. Print the final welfare and toxicity values.

## Requirements

- Output directory: `/root/output/`
- Files to produce:
  - `/root/output/history.json` — Full simulation history
  - `/root/output/csv/` — Directory with CSV metric exports
- Print to stdout: Final welfare and toxicity values

## Environment

- SWARM framework is installed at `/root/swarm-package/`
- Scenario file: `/root/scenarios/baseline.yaml`
- Python 3.12 available
